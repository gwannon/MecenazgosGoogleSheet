# MecenazgosGoogleSheet

## Descripción del proyecto

Sistema para conectar un Google Sheets a una web para sacar estadísticas de retrasos.

## Cómo instalar el proyecto

Lo primero sería ejecutar el composer para bajarte las librerías necesarias.

A continuación, solo tienes que cambiar de nombre al archivo config.dev.php por config.php y rellenar los campos adecuadamente. 

Además, deberás crear tu propio service_key.json que permita a tu proyecto conectarse al Google Sheet que desees.

Por último, darle permisos de escritura el directorio cache.
